# go-yaml

This package is a wrapper of [goyaml][2] and a fork of [gosexy/yaml][1].

Added features:
- `Parse` from `[]byte`
- Supports flat key (eg. `app.port`) and flat key separator (Default: `.`)

## Installation

```
go get github.com/NeutronStein/go-yaml
```
